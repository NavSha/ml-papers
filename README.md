# ML Papers — The 14 That Matter

A curated project around the most important machine learning / AI papers of the last ~15 years, filtered through one lens: **what every engineer, technical PM, and founder should know** — not citation counts, but what changes how you build, prioritize, or place bets today.

The end goal is to make these papers as accessible as possible to that audience — likely as an interactive site (timeline spine + per-paper cards/widgets, Pudding-style).

## Contents

- [PAPERS.md](PAPERS.md) — the final curated list of 14 papers, with the arc, the "consider later" pile, and what was cut and why
- [VISUALIZATIONS.md](VISUALIZATIONS.md) — the best existing visual explainers of ML papers (the bar to beat), what makes them work, and the tiered format plan for our version

## Origin

Curated interactively in a Claude Code session (June 20, 2026): started from a top-20 list (2015–2025), extended back to 2009, then went paper-by-paper with keep/drop/maybe votes, and finally applied the engineer/PM/founder lens to land on 14.
